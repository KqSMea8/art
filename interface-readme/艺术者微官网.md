# 艺术者微官网
## 1 画作
### 1.1 画作详情
> 请求方法：post
> 需要登录：是
> 请求地址：https://test-api.artzhe.com/M/Artwork/getArtDetail
> 需要token：是(GET方式传token)

| --------   | :-----:  | :----:  | :----:  | :----:  |:----:|
|id|艺术品id|int|是|-|-|
